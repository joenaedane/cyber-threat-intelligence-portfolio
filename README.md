---

## Portfolio Approach

This portfolio is organized to reflect how modern security teams operate across multiple analytical layers:

* **Lab Environment:** attacker capability simulation and technical validation
* **Detection Engineering:** development of telemetry-driven detection logic
* **Threat Hunting Playbooks:** hypothesis-driven investigative workflows
* **Campaign Analysis:** intelligence synthesis connecting multiple suspicious behaviours into coherent intrusion narratives
* **Intelligence Reports:** strategic analysis of attacker tradecraft and threat trends

Each campaign phase demonstrates how individual alerts or technical findings can be connected into broader intrusion hypotheses.
