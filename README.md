## Campaign Simulation (Core of Portfolio)

This portfolio is built around a **multi-phase enterprise intrusion simulation**:

```plaintext
Initial Access → Credential Access → Lateral Movement → Domain Control
→ Persistence → Command & Control → Data Exfiltration → Ransomware Impact
```

* Credential dumping and Pass-the-Hash lateral movement
* Active Directory privilege escalation and domain control
* Persistence via service account manipulation
* Command-and-control (C2) via scheduled task beaconing
* Low-and-slow data exfiltration using native tools
* Ransomware-style impact via file modification and recovery inhibition
